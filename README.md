# Udacity Project: Readable 

React / Redux project for React Nanodegree from Udacity.

## Server

```sh
git clone https://github.com/joseasousa/readable.git
cd readable/api-server
yarn
yarn start
```

## App
```sh
git clone https://github.com/joseasousa/readable.git
cd readable/client
yarn
yarn start
```
